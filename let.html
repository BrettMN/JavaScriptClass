let

What?

The let statement declares a block scope local variable, optionally initializing it to a value.

let does not hoist.

When?

Can be used in place of var when you want block scoping.

How?

Scoped to the brackets:

let foo = 2;
if(true){
	let bar = 1;
}
console.log( foo + bar );

Does not hoist and Temporal Deadzone:

function doSomething(){
	console.log(a);  //Error?
	let a = 1;
	console.log(a);
}
