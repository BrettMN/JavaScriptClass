set

What?

The set object lets you store unique values of any type, whether primitive values or object references.

When?

Can be used in place of var when you want the value to stay constant though out the use of your program.

How?

Basic set:

var set = new Set();
set.add(1);
set.add(2);
set.add(3);
console.log(set.size); //3

Unique Values:

var set = new Set();
set.add(1);
set.add(1);
set.add(1);
console.log(set.size);  //1

No Typecasting:

var set = new Set();
set.add(1);
set.has(1); //true
set.has("1"); //false
set.add("1");
set.has("1"); //true

Iteration:

var items = new Set([1, 2, 3, 4, 5]);
for (let num of items) {
	console.log(num); //logs 1, 2, 3, 4, 5
}
